The MoonPay On-Ramp is a standalone widget product with an optional API and webhooks for an enhanced user experience. We provide web and mobile SDKs to render the widget in your app and a backend Node SDK for performing server-side functions.

# Components for on-ramp integrations

| Component                                          | Features                                                                                                                                                                                                                    |
| -------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [URL integration](/v1.0/docs/on-ramp-overview)     | An alternative to using our SDKs Generate MoonPay widget links to show the widget in an iframe or mobile webview                                                                                                            |
| [Web SDKs](/docs/on-ramp-web-sdk)                  | [Parameters](/docs/on-ramp-configure-user-journey-params) to pre-fill cryptocurrency, amount, user email, etc. Includes [communication layer](/docs/on-ramp-sdk-events-details) between partner apps and the MoonPay widget |
| [Mobile SDKs](/docs/on-ramp-react-native-sdk)      | React Native SDK                                                                                                                                                                                                            |
| [Node SDK](/docs/on-ramp-node-sdk)                 | Perform server-side functions, like [URL signing](/docs/on-ramp-enhance-security-using-signed-urls)                                                                                                                         |
| [MoonPay API](/reference)                          | Get minimum / maximum purchase amounts Get buy quote Show transaction history                                                                                                                                               |
| [Webhooks](/reference/reference-webhooks-overview) | Get transaction updates                                                                                                                                                                                                     |
