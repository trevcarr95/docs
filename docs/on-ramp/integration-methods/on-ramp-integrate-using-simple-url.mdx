You can use the below pre-writsdfsdften code to quickly get started on adding a '**Buy Crypto**' button that opens the MoonPay widget in a new browser tab.

**Sandbox**: [https://buy-sandbox.moonpay.com/?apiKey=pk_test_123](https://buy-sandbox.moonpay.com/?apiKey=pk_test_123)

**Production**: [https://buy.moonpay.com/?apiKey=pk_live_INSERT_API_KEY](https://buy.moonpay.com/?apiKey=pk_live_INSERT_API_KEY)

<CodeGroup>
  ```html On-ramp.html
  <!--Opens the MoonPay widget in a new browser tab-->
  <body>
      <div class="container">
          <button onclick="window.open('https://buy-sandbox.moonpay.com/?apiKey=pk_test_123', '_blank')">Buy with MoonPay</button>
      </div>
  </body>
  ```
