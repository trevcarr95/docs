The MoonPay Off-Ramp is a standalone widget product with an optional API and webhooks for an enhanced user experience. We provide web and mobile SDKs to render the widget in your app and a backend Node SDK for performing server-side functions.

# Components for on-ramp integrations

| Component                                                    | Features                                                                                                                                                                                                                           |
| ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [URL integration](/docs/off-ramp-integrate-using-simple-url) | An alternative to using our SDKs Generate MoonPay widget links to show the widget in an iframe                                                                                                                                     |
| [Web SDKs](/docs/off-ramp-web-sdk)                           | [Parameters](/docs/ramps-sdk-sell-params) to pre-fill cryptocurrency, fiat currency, crypto amount, user email, etc. Includes [communication layer](/docs/off-ramp-sdk-events-details) between partner apps and the MoonPay widget |
| [Mobile SDKs](/docs/off-ramp-react-native-sdk)               | React Native SDK                                                                                                                                                                                                                   |
| [Node SDK](/docs/off-ramp-node-sdk)                          | Perform server-side functions, like [URL signing](/docs/off-ramp-enhance-security-using-signed-urls)                                                                                                                               |
| [MoonPay API](/reference/getsellquote)                       | Get minimum / maximum sell amounts Get sell quote Show transaction history                                                                                                                                                         |
| [Webhooks](/reference/reference-webhooks-overview)           | Get transaction updates                                                                                                                                                                                                            |
