## Supported payment method and fiat combinations

Today, MoonPay supports a number of push payment methods into Virtual Accounts.

| **Payment Method**  | Currency | Availability |
| ------------------- | -------- | ------------ |
| **ACH (Same Day)**  | USD      | ✅           |
| **FedWire**         | USD      | ✅           |
| **SEPA Instant**    | EUR      | ✅           |
| **Faster Payments** | GBP      | ✅           |

If you are looking to support other methods, such as cards, Apple Pay, Google Pay, PayPal, Venmo, Pix then [our On-Ramp](/docs/on-ramp-overview) product might be a better fit for your team.
