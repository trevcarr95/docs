---
title: "List Buy transactions"
description: "get https://api.moonpay.com/v1/transactions"
---

Returns an array of successful Buy transactions which fulfill criteria supplied in the query parameters. Each entry in the array is a separate transaction object. Transactions will be listed from newest to oldest.

Complete list of transaction object parameters and descriptions can be found [here](https://moonpay.readme.io/reference/getbuytransaction).
